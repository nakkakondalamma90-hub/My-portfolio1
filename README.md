# Personal Portfolio Website

## 📌 Project Overview

This project is a fully responsive personal portfolio website developed using HTML, CSS, and Bootstrap. The website showcases personal information, technical skills, projects, and contact details in a clean and professional layout.

The portfolio is designed to work seamlessly across desktops, tablets, and mobile devices.

---

## 🚀 Features

- Responsive design for all screen sizes
- Fixed navigation bar with smooth scrolling
- Home section with introduction
- Skills section showcasing technical skills
- Projects section displaying completed works
- Contact section with personal details
- Modern UI using Bootstrap 5
- Easy to customize and deploy

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Bootstrap Components)

## 📖 Deployment on GitHub Pages

1. Push the project to GitHub.
2. Open the repository.
3. Go to **Settings**.
4. Select **Pages** from the left menu.
5. Under **Source**, choose:
   - Branch: `main`
   - Folder: `/ (root)`
6. Click **Save**.

Your website will be available at:

```text
https://yourusername.github.io/portfolio-website/
```
